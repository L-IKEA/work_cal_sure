# 슈어소프트 근태 연동 유연근무 계산기 (Work Planner - Suresoft Attendance Sync)

본 프로젝트는 주간 목표 근무 시간을 충족하기 위해, 실제 근태 실적(출퇴근 시각)과 연차/휴일 데이터를 바탕으로 잔여 근무일의 권장 근무 일정 및 최적 퇴근 시각을 자동으로 산출 및 시뮬레이션해 주는 스마트 웹 어플리케이션이자 크롬 확장 프로그램입니다.

---

## 🌟 주요 기능

1. **사내 근태 데이터 자동 연동**
   * 크롬 확장 프로그램(Manifest V3)이 내장되어 있어, 사내 근태관리 사이트(EHR) 접속 시 백그라운드에서 `/month` API 통신 데이터를 안전하게 가로채 로컬 브라우저 저장소(`chrome.storage.local`)에 자동으로 동기화합니다.
   * 계산기 앱을 열어둔 채 근태 사이트만 조회해도 화면 내 데이터가 실시간 비동기 갱신됩니다.

2. **세분화된 출/퇴근 동기화 체크박스 (데이터 유실 방지)**
   * 요일별로 출근 시각과 퇴근 시각의 연동 여부를 개별 체크박스로 제어할 수 있습니다.
   * 목요일 오전에 출근하여 동기화할 때, **출근만 체크하여 동기화**하면 기존에 사용자가 기입해 둔 퇴근 계획 시각(예: 18:00 퇴근 예정)과 고정/예측 상태가 초기화되지 않고 안전하게 보존됩니다.

3. **하이브리드 휴일 가산 규칙 지원**
   * 달력 상 빨간 날(법정공휴일)뿐만 아니라, **근로자의 날(5월 1일)이나 대체휴무 등 평일 상의 사내 고정 휴일** 규격도 자동으로 감지하여 `🎌 휴일`로 매핑합니다.
   * 휴일 지정 시 하루 8시간의 근무 시간이 자동으로 가산 인정됩니다. (반차는 4시간 가산)

4. **스마트 유연근무 배분 알고리즘**
   * **동일한 순근무 시간 배정 (`sameWorkTime`)**: 이진 탐색(Binary Search / Water-filling) 알고리즘을 구동하여 남은 잔여일 동안 매일 동일하게 근무해야 할 최소 순근무 시간을 수렴 계산합니다. (반차일이 있으면 차등 반영)
   * **동일한 퇴근 시각 배정 (`sameClockOut`)**: 매일 출근하는 시간이 조금씩 다르더라도, 잔여일 모두 동일한 시각에 퇴근하여 목표 시간을 완벽히 채울 수 있는 퇴근 시각을 역산합니다.

5. **수동 입력 Fallback 지원**
   * 확장 프로그램이 미설치된 브라우저 환경에서도 개발자 도구(F12) Network 탭에서 복사한 JSON 본문을 붙여넣는 방식으로 동일한 연동 기능을 지원합니다.

## 📂 프로젝트 구조

```text
├── dist/                  # 빌드 완료 시 생성되는 크롬 확장 프로그램 로드용 폴더
├── public/                # 크롬 확장 프로그램 정적 에셋 및 백그라운드 스크립트
│   ├── manifest.json      # 확장 프로그램 설정 명세서 (Manifest V3)
│   ├── background.js      # 확장 프로그램 아이콘 클릭 핸들러 (서비스 워커)
│   ├── content-ehr.js     # 사내 근태 웹페이지 Fetch/XHR 데이터 캡처용 MAIN World 스크립트
│   └── content-bridge.js  # 가로챈 데이터를 크롬 로컬 저장소로 교환하는 브릿지 스크립트
├── src/                   # React 컴포넌트 및 핵심 비즈니스 로직
│   ├── App.tsx            # 메인 시뮬레이터 뷰, 연동 팝업, 이진 탐색 분배 알고리즘 구현
│   ├── main.tsx           # React 엔트리 포인트
│   └── index.css          # Tailwind CSS v4 스타일시트
├── tsconfig.json          # TypeScript 컴파일 옵션
├── vite.config.ts         # Vite 빌드 설정
└── package.json           # 종속성 라이브러리 및 빌드 스크립트
```

## 🛠️ 설치 및 사용 방법

### 1단계: 프로젝트 빌드
로컬 환경에서 개발을 시작하거나 확장 프로그램 폴더를 추출하려면 프로젝트 패키지를 설치하고 빌드를 진행합니다.
```bash
# 종속성 라이브러리 설치
npm install

# 프로덕션 빌드 (dist 폴더 생성)
npm run build
```

### 2단계: 크롬 확장 프로그램 등록
1. 크롬 브라우저를 열고 주소창에 `chrome://extensions/`를 입력해 이동합니다.
2. 우측 상단의 **개발자 모드 (Developer mode)** 스위치를 활성화합니다.
3. 좌측 상단의 **압축해제된 확장 프로그램을 로드합니다 (Load unpacked)** 버튼을 클릭합니다.
4. 본 프로젝트 루트 하위에 빌드된 **`dist`** 폴더를 선택하여 등록합니다.

### 3단계: 근태 데이터 자동 연동 시작
1. 사내 그룹웨어 근태관리 페이지([https://gw.suresofttech.com/app/ehr](https://gw.suresofttech.com/app/ehr))에 로그인하여 접속합니다. (접속 시 확장 프로그램이 API 응답을 실시간 자동 캡처하여 로컬에 저장합니다.)
2. 브라우저 툴바에서 등록한 확장 프로그램 아이콘을 클릭하면 **유연근무 계산기** 탭이 새로 열립니다.
3. 계산기 우측 상단의 **근태 데이터 연동** 버튼을 눌러 연동 주차를 선택하고 데이터를 시뮬레이터에 적용합니다.

## 💻 로컬 개발 서버 구동
코드 수정 사항을 실시간 브라우저로 핫 리로딩(Hot Reloading)하며 디버깅하려면 개발 서버를 구동합니다.
```bash
npm run dev
```

## 📝 라이선스 및 주의사항
* 본 도구는 개인 편의를 위해 개발된 근무 시간 보조 시뮬레이터입니다. 공식 근태 데이터와 계산기의 오차(단위 절사 등)가 있을 수 있으므로 최종 검토용으로 활용하시길 권장합니다.
* 가로채는 모든 근태 데이터는 외부 서버로 전송되지 않으며, 사용자 본인의 크롬 브라우저 로컬 저장소(`chrome.storage.local` 및 `localStorage`)에만 안전하게 보관됩니다.
